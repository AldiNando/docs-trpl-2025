## Software Requirement
## 1. Aplikasi Web

Untuk membangun web-aplikas saya memerlukan beberapa teknologi dasar yang dapat menunjang fungsionalitas aplikasi secara optimal.

## 1.1 Web Server

Aplikasi web ini akan di-hosting pada Apache HTTP Server atau Nginx. Web server ini bertugas untuk melayani permintaan dari pengguna yang mengakses aplikasi melalui browser, seperti halaman HTML, CSS, dan JavaScript. Dengan menggunakan server ini, aplikasi dapat berjalan dengan lancar dan responsif.

## 1.2 Database Management System

Untuk menyimpan berbagai data penting, saya akan menggunakan MySQL atau MariaDB sebagai sistem manajemen database. Keduanya adalah pilihan yang sangat baik karena sering digunakan dalam aplikasi berbasis PHP dan menawarkan fitur yang stabil dan efisien untuk pengelolaan data.

## 1.3 Backend Framework

Di sisi server, aplikasi ini akan dikembangkan menggunakan PHP 7.4+. PHP berfungsi untuk memproses logika server-side, seperti memvalidasi input pengguna, mengelola data, Versi PHP yang lebih baru memastikan aplikasi berjalan lebih cepat dan lebih aman.

## 1.4 Frontend Technology

Antarmuka pengguna akan dibangun menggunakan HTML5, CSS3, dan JavaScript, yang merupakan dasar dalam pembuatan aplikasi web modern. Untuk memberikan pengalaman pengguna yang lebih responsif dan mobile-friendly, kita akan menggunakan Bootstrap 4/5 & Tailwind, sebuah framework CSS yang sangat populer. Selain itu, jQuery akan digunakan untuk menambah interaktivitas, seperti deteksi lokasi secara otomatis dan pengiriman notifikasi darurat.

## 1.5 Development Tool
Dalam pengembangan aplikasi ini, kami akan menggunakan Visual Studio Code sebagai text editor utama untuk menulis kode aplikasi. Untuk pengembangan lokal, kita akan memanfaatkan XAMPP atau MAMP, yang menyediakan server lokal yang dapat menjalankan aplikasi berbasis PHP dengan mudah.

Dengan menggunakan kombinasi teknologi ini, aplikasi web pelacak lokasi teman dalam keadaan darurat akan memiliki performa yang optimal, responsif, dan mudah digunakan.
